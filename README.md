# Node console note app
 
 Console note taking app made with node js. You can create notes,read notes,list notes and remove notes with basic terminal commands. Created notes keep with JSON file and you can modify this JSON file via command line.
 
 Use with node app.js [command]
 
 Command List:
 
 Commands:
  app.js add     Add note
  app.js remove  Remove a note
  app.js read    Read a note
  app.js list    Listing all the notes

Options:
  --help     Show help                                                 
  --version  Show version number
